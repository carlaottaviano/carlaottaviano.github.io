Aplicación para dispositivos
Android y IOS

27 Sep

Premedic Movil está disponible para dispositivos Android & Iphone. Cuenta con diversas funcionalidades: Geolocalización personalizada según el plan del cliente, teléfonos de emergencia para facilitar el uso de la medicina prepaga cuando más se necesita y mapa para saber cómo llegar al destino. Es una aplicación fácil de usar, intuitiva y ecológica.

La aplicación se puede descargar desde el PlayStore de Android o el App Store de Apple.
